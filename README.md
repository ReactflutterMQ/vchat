#Project Introduction
This is a desktop chat robot based on the vue3+Electron+Ts technology stack and integrating multiple AI large models (ERNIE Bot/Tongyi Qianwen/Claude/Deepseek). It allows users to ask questions to the application and also enables the application to generate code. Additionally, the project utilizes Cursor for auxiliary development, achieving twice the result with half the effort and significantly improving development efficiency

#How to use
1、Use the command `git clone repository_address` to clone the project locally
2、npm install to install project dependencies
3、npm run start to run the project
